# App-Personal-Key-Indicators-of-Heart-Disease
   ## You can estimate your chance of getting  heart disease in seconds!
        The app is built using the 2020 CDC annual survey data of 400,000 adults, focusing on their health status. It leverages a machine learning algorithm, logistic regression, achieving an accuracy of 88%.

           For more information about the dataset, visit Key Indicators of Heart Disease. If you’d like to explore the app on Streamlit, you can access it here: Streamlit App.

      Note: These results are not a substitute for medical diagnosis!
